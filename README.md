PowerFox is a secure, modern, and open-source web browser for Mac OS X 10.6 Snow Leopard, based on the open-source UXP codebase.

For more information, visit the [PowerFox website](https://powerfox.jazzzny.me).

## Source Code
The source tree will be imported to this repository after it is appropriately cleaned up and all desirable additions are submitted upstream.

In the meanwhile, every release contains archives of the `platform` and `base` source trees.
